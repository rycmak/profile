Air Galaxy Project

An AirBnB-style Ruby-on-Rails app built during a super-fun fullstack web-development bootcamp (Le Wagon) that I attended.
Instead of homes, we travelled a few centuries ahead and offered galaxies for rent! Functionalities include the ability to log in, book a galaxy or list one of your own (if you're lucky enough to own such a celestial object), and view your dashboard.

BuddyMeUp Project

BuddyMeUp is an application built with IsabellaHoesch for the non-profit organization Women Who Code to pair particularly female coders from around the world to work on fun projects and practice programming together.
Built on Streamlit, the app collects participants' information, which is fed to a custom pair-matching algorithm to optimize matches based on stated preferences, goals, and other descriptors.
