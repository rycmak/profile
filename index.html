Project

BuddyMeUp Project

BuddyMeUp is an application built with IsabellaHoesch for the non-profit organization Women Who Code to pair especially female coders from around the world to work on fun projects and practice programming together.
Built on Streamlit, the app collects participants' information, which is fed to a custom pair-matching algorithm to optimize matches based on stated preferences, goals, and other descriptors.
