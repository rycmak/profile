BuddyMeUp Project

Working with IsabellaHoesch, we built an application called BuddyMeUp for the non-profit Women Who Code to pair coders from around the world to work on fun projects and practice programming together.
Built on Streamlit, the app collects participants' information, which is fed to a custom pair-matching algorithm to optimize matches based on stated preferences, goals, and self-descriptions.
